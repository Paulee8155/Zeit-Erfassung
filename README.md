# Zeit-Erfassung

Digitales Zeiterfassungs- und Leistungsnachweissystem für Kran- und LKW-Betriebe.

## Funktionen

- **Arbeitszeiterfassung** – Beginn, Ende, Pausen, Tätigkeiten (Auf-/Abbau Kran)
- **Leistungsnachweise** – Kundenunterschrift, Auftragsdaten
- **Wochenberichte** – Prüfung, Genehmigung durch Dispo/Personal
- **Lohnerfassung** – Spesen, Erschwerniszulagen, Prämien, Überstunden
- **Export** – Vorbereitung für Lodas

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (State Management)
- date-fns
